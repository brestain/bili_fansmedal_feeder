### 简介

2025年08月12日, b站粉丝牌(粉丝勋章)新规则:
https://www.bilibili.com/opus/1100097465625870441  
 

省流: 不论是否上舰, 勋章等级都可以从1级升到120级  
白嫖能获取经验的方式:  
1. 每日每观看五分钟可获得6亲密度，最多可获得30亲密度(必须主播开播)
2. 给视频投币：1硬币=10亲密度，每日上限100亲密度


本项目做什么: 帮用户自动实现每日观看五分钟直播, 获得亲密度, 自动升级勋章. 自动从有粉丝勋章的已开播的经验值没刷满的直播间观看直播,直到刷满为止. 
(小字)由于必须主播开播才能获取亲密度, 而主播可能一整天都不开播, 所以本脚本不是定时每天执行, 而是一直在后台运行观察有没有需要刷的直播间.

### 注意事项
如果存在多个正在直播, 观看直播的优先级顺序: 
在fansmedal_weight.yaml中配置的权重越大越优先
相同权重的,粉丝牌等级越高越优先

注意: 每个粉丝勋章刷满经验需要25分钟, 理论上一天最多刷满57个勋章, 但是每个主播的直播时间有限而且会在晚上集中, 所以实际上能刷满的勋章数量会更有限. 建议在fansmedal_weight.yaml中给想要优先获得亲密度(尤其是直播时间比较短的)的直播间配置更高的权重
