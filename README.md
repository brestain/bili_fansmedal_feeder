## 📖 简介 <a id="简介"></a>

### 新规则说明

**2025年08月12日**，B站粉丝牌（粉丝勋章）新规则发布：  
🔗 [查看官方公告](https://www.bilibili.com/opus/1100097465625870441)

### 白嫖获取经验方式

1. **观看直播**：每日每观看五分钟可获得 6 亲密度，最多可获得 30 亲密度（**必须主播开播**）
2. **视频投币**：1 硬币 = 10 亲密度，每日上限 100 亲密度

### 项目功能

本项目帮助用户**自动实现每日观看直播**，获得亲密度，自动升级勋章。


> 💡 **提示**：由于必须主播开播才能获取亲密度，而主播可能一整天都不开播，所以本脚本不是定时每天执行，而是**一直在后台运行**。

## ⚠️ 注意事项 <a id="注意事项"></a>

### 观看优先级

如果存在多个正在直播的直播间，观看直播的优先级顺序：

1. **权重优先**：在 `fansmedal_weight.yaml` 中配置的权重越大越优先
2. **等级优先**：相同权重的，粉丝牌等级越高越优先

### 效率说明

- ⏱️ 每个粉丝勋章刷满经验需要 **25 分钟**
- 📊 理论上一天最多刷满 **57 个勋章**
- ⚠️ 实际上能刷满的勋章数量会更有限，因为：
  - 每个主播的直播时间有限
  - 直播时间会在晚上集中

### 配置建议

建议在 `fansmedal_weight.yaml` 中给想要优先获得亲密度的直播间配置更高的权重，尤其是：
- 直播时间比较短的直播间
- **你特别关注的 UP 主**

- 🤖 **自动化运行**：无需手动操作，自动观看直播获取亲密度

- 🔄 **持续监控**：后台持续运行，实时检测开播状态
- ⚙️ **灵活配置**：支持权重配置、白名单、黑名单等个性化设置
